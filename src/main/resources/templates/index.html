Charging Stations Service

List Stations

The List Stations operation returns a list of all available charging stations.

Request

HTTP Method: GET
Endpoint: /stations

Response

HTTP Status Code: 200 OK
Content-Type: application/json
Body: 

{
  "stations": [
    {
      "id": 1,
      "name": "Station A",
      "address": "123 Main St",
      "location": {
        "type": "Point",
        "coordinates": [-122.6765, 45.5231]
      },
      "amenities": "WiFi, Coffee shop"
    },
    {
      "id": 2,
      "name": "Station B",
      "address": "456 Second St",
      "location": {
        "type": "Point",
        "coordinates": [-122.6816, 45.5237]
      },
      "amenities": "Restroom, Parking"
    }
  ]
}

Find Stations Near a Location

The Find Stations Near a Location operation returns a list of charging stations that are within a certain distance of a given location.

Request

HTTP Method: GET
Endpoint: /stations/nearby?lat={latitude}&lon={longitude}&distance={distance}
Query Parameters:
- lat (required): latitude of the location
- lon (required): longitude of the location
- distance (optional): search radius in meters, default is 1000 meters

Response

HTTP Status Code: 200 OK
Content-Type: application/json
Body: 

{
  "stations": [
    {
      "id": 1,
      "name": "Station A",
      "address": "123 Main St",
      "location": {
        "type": "Point",
        "coordinates": [-122.6765, 45.5231]
      },
      "amenities": "WiFi, Coffee shop"
    }
  ]
}

Run SQL

The Run SQL operation creates the database schema and inserts seed data into the `stations` and `connector` tables.

Request

HTTP Method: GET
Endpoint: /runsql

Response

HTTP Status Code: 200 OK
Content-Type: application/json
Body: 

{
  "message": "Schema and seed data have been created successfully."
}
